# BangumiScripts

适用于 [Bangumi](https://bgm.tv) 的用户脚本

## 使用方法

1. 安装[油猴脚本](http://tampermonkey.net/)
2. 导入相应用户脚本

### Tell Me More

用于在动画排行，目录，个人收藏页面展示更多信息。 安装：[GreasyFork](https://greasyfork.org/scripts/37629-tell-me-more)
